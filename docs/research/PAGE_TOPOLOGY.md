# Page Topology — csboyasizgocukduzeltme.com

WordPress 5.8.5 + Elementor + Autrics theme. Turkish single-page car dent repair business.

## Sections (top → bottom)

1. **Header / Top Bar + Main Nav** — sticky logo + nav
2. **Hero Slider** — 3 slides (Owl Carousel)
   - Slide 1: "Memnuniyet Garantisi! / Profesyonel Ekip"
   - Slide 2: "Yüzlerce Mutlu Müşteri! / Referans İşlerimizi İnceleyin!"
   - Slide 3: "CS BOYASIZ GÖÇÜK ONARIM MERKEZİ / Aracınız Orijinal Kalsın!"
3. **Neden Biz?** — title + 4 feature icon cards (Profesyonel Ekip, %100 Sonuç, Müşteri Memnuniyeti, Detaylı Onarım)
4. **Çalışma Süreci** — title + 3 process steps (alternating bg) with car/tools/smile icons
5. **Hizmetlerimiz** — title bar with bg image
6. **Service Carousel** — 5 services in Owl Carousel (Far Temizleme, Pasta & Cila, Alüminyum Parça Düzeltme, Dolu Hasarı Onarımı, Boyasız Göçük Onarımı)
7. **Bizden Haberler** — blog title + 1 post card
8. **Contact CTA** — address, phone, WhatsApp (3 columns)
9. **Footer** — about + service links + recent post + copyright
10. **Floating WhatsApp button** (mobile)

## Tech
- Roboto + Dosis (Google Fonts)
- Owl Carousel for sliders
- Custom icon font (`icon-user`, `icon-star`, `icon-smile`, `icon-tools`, `icon-car`)
- FontAwesome for social/arrows
- jQuery + Elementor
- No smooth scroll lib

## Interaction model
Mostly static. Two carousels: hero (3 slides, auto-advance) and services (5 slides, navigation arrows).
