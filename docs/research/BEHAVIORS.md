# Behaviors

- **Hero slider** — 3 slides with dark overlay over background image. Auto-advances. Has prev/next arrows + dots.
- **Service carousel** — 5 cards, manual nav arrows, fades on hover (red overlay reveal title).
- **Header** — fixed/sticky top with logo + nav links. White bg.
- **Buttons** — bg `#ee3131`. Hover: opacity/scale subtle.
- **Process steps** — middle card has red bg variant (`bg-red` class), others white. Decorative shapes left/right.
- **Footer** — dark navy `#06163a`, lighter copyright bar `#2f3967`.
- **Floating mobile CTA** — fixed bottom phone button on viewport ≤650px.
- **Scroll-to-top** — bottom-right floating button.

## Responsive
- Desktop ≥992: 4-col Why Us, 3-col process, multi-card service carousel.
- Tablet 768-991: 2-col Why Us, stacks process.
- Mobile <768: single column, hamburger nav, fixed phone CTA.
