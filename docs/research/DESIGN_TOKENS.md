# Design Tokens

## Colors
- Primary red: `#ee3131` (rgb 238,49,49) — buttons, accents, hover bg
- Red hover: `#dd3333`
- Dark navy: `#06163a` — footer bg
- Mid navy: `#2f3967` — copyright bar
- Body text: `#101010`
- Muted text: `#666666`
- Light text: `#838b9d`
- White: `#ffffff`
- Overlay black: `rgba(0,0,0,0.5)`

## Fonts
- **Roboto** — body, h1, h2, p (weights 400, 600, 700)
- **Dosis** — h3 / card titles (weight 700)
- Both Google Fonts

## Typography (computed)
- h1: Roboto 36px / 600 / line-height 43.2 / uppercase / white
- h2: Roboto 30px / 600 / capitalize / `#101010`
- h3: Dosis 20px / 700 / `#101010`
- p: Roboto 16-20px / 400-600 / line-height 24-30
- a: Roboto 16/400 / `#101010`
- body: Roboto 16/400 / `#666`

## Buttons
- Primary: bg `#ee3131`, color white, padding ~14px 32px, uppercase, rounded ~3px
- Hover: slight darken / scale

## Borders/Radius
- Cards: 0 (square) or 4px
- Buttons: ~3px

## Icons
Custom icon font (based on Autrics theme):
- icon-user, icon-star, icon-smile, icon-tools, icon-car
Plus FontAwesome v4: fa-instagram, fa-facebook, fa-twitter, fa-whatsapp, fa-angle-left/right, fa-angle-double-up
